> **개발기간: 2023.02 ~ 2023.3**

## 프로젝트 소개

은행 서버의 API들을 구현해본 토이 프로젝트입니다.
- 회원가입
- 친구 등록
- 친구 조회
- 계좌 조회
- 계좌 이체
